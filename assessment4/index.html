CSS can be added to HTML documents in three ways

  • inline - by using the style attribute inside HTML element
  • External - by using a link element to link to an external CSS file.
  • internal - ny using a style element in the head section
  • The most common way ta add CSS. is to keep the styles in external css files. however, in this tutorial we will use inline and internal style, because this is easier to demonstrate, and easier for you to try it yourself

When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple elements. ID,on the other hand,applies a style to a unique element.ID is also special in that you acn use a special URL to link directly to an element and it's used by JavaScript

in CSS, selector are used to target a specific element or range of elements on a webpage. Once an element has been targeted, a style or set of style can be applied to the element

there is a wide range of selector available/ two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied
